# Para ter acesso ao projeto de forma visual siga o seguinte passo a passo: 
1. instale o servidor local: xampp
2. baixe o projeto: git clone https://github.com/ludmillasophia/portfoliohub.git
3. copie o projeto dentro de xampp/httdocs
4. abra o navegador e cole a seguinte url http://localhost/projeto
