# Código de Conduta

## Nosso Compromisso

Este projeto tem como objetivo promover um ambiente respeitoso, colaborativo e inclusivo para todos os participantes.

## Comportamentos Esperados

* Respeitar todos os colaboradores.
* Utilizar linguagem profissional e cordial.
* Aceitar críticas construtivas.
* Compartilhar conhecimento de forma colaborativa.
* Agir de maneira ética e responsável.

## Comportamentos Inaceitáveis

* Assédio, discriminação ou ofensas.
* Linguagem abusiva ou desrespeitosa.
* Divulgação de informações privadas sem autorização.
* Qualquer atitude que prejudique a colaboração entre os participantes.

## Aplicação

Os mantenedores do projeto têm a responsabilidade de garantir o cumprimento deste código de conduta e poderão tomar medidas apropriadas diante de comportamentos inadequados.
