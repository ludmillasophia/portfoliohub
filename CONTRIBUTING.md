# Guia de Contribuição

Obrigado por contribuir com o PortfolioHUB!

## Como reportar problemas

1. Acesse a aba Issues.
2. Descreva o problema encontrado.
3. Inclua passos para reproduzir o erro.

## Como contribuir

1. Crie uma nova branch.
2. Realize as alterações necessárias.
3. Faça commit das alterações.
4. Abra um Pull Request para revisão.

## Boas práticas

* Escrever código organizado.
* Documentar alterações importantes.
* Respeitar os demais colaboradores.
